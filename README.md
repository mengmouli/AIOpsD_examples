# Setup Instructions

If you are already familiar with Python virtual environments,  
you may create and activate `.venv` in your preferred way.

Otherwise, follow the instructions below.

Set it up before running the code.

---


1. Open your Python IDE (e.g., VS Code) and open a terminal  
   (or open Terminal and navigate to this folder)


2. Run the following script:
## Windows 
 ```PowerShell
python3 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## macOS / Linux
 ```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
